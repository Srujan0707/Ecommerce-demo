1) First Create New Component for the Reative Form - Restuarant Form
2) import "ReactiveFormsModule" from angular/forms in app.module.ts file at the same time we need to place the ReactiveFormsModule in import section.
3) If we want to create "Reactive-Form" in Angular then we need two classes : 1:- FormGroup 2:- FormControl from "@angular/forms"
4) Now we are going to create a instance or object for this class.
RestuarantForm = new FormGroup(); //So RestuarantForm is a object of FormGroup.
5) Now in the constructor of FormGroup we have to pass an object.., Inside this object we are giving Input Elements as a FormControl.
6) Defualt Value or Validator should be add for Input Element, for that we need to import this validator for ex: import "Validator" from angular/forms.
7) Now this Restuarant Form should be Bind in Our HTML Template.
8) Open the Browser to see the Reactive-Form.
